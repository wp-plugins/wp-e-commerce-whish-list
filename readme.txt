== Installation ==
To use this plugin, you will need:
* 	an installed and configured copy of [WordPress][]

	(version 4.0 or later).
*	FTP, SFTP or shell access to your web host

	1. Add eCommerce Wish List Widget from your theme
	
== Frequently Asked Questions ==

1. Is this plugin working without WP eCommerce Plugin?

	No, WP eCommerce Plugin is required for this plugin



= New Installations =

Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin


Method-B:

1.	Download the WP eCommerce Wishlist installation package and extract the files on

	your computer. 
2.	Create a new directory with the name of this plugin in the `wp-content/plugins`

	directory of your WordPress installation. Use an FTP or SFTP client to

	upload the contents of your WP eCommerce Wishlist archive to the new directory

	that you just created on your web host.
3.	Log in to the WordPress Dashboard and activate this plugin.
4.	Once the plugin is activated, a new **WP eCommerce Wishlist** sub-menu will appear in your Wordpress admin -> settings menu.

== Upgrade Notice ==


= Upgrades =

To *upgrade* an existing installation of this plugin to the most recent release:

1.	Download the plugin installation package and extract the files on

	your computer. 
2.	Upload the new PHP files to `wp-content/plugins/wp-e-commerce-wish-list`,

	overwriting any existing wp-e-commerce-wish-list files that are there.

	

3.	Log in to your WordPress administrative interface immediately in order

	to see whether there are any further tasks that you need to perform

	to complete the upgrade.
4.	Enjoy your newer and hotter installation of wp-e-commerce-wish-list
